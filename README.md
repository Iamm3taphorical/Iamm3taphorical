- **Data Structures & Algorithms (Java):** deepening mastery with production-quality implementations and documented learning resources (see `CSE220_Data_Structure` for curated examples and exercises).

- **Model Deployment & Cloud Basics:** packaging models, exposing inference endpoints, and experimenting with lightweight inference on the edge and in cloud environments (AWS Cloud Foundations background helps accelerate this).  
- **Open Source & Teaching:** publishing clear, well-documented learning repos (lecture notes, lab resources) to help peers learn data structures, object detection, and robotics integration.
